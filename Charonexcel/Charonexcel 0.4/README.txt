CharonExcel v. 0.4

[RU]

Программа предначзначена для работы с большим объёмом данных в таблицах Excel.
Её функция заключается в распределении данных из фида в шаблоны по категориям в соотвутствующие столбцы.

Принцип работы можно увидеть на примере работы программы CharonExcel 0.4.exe с файлами из папки excels. Фид (файл A.xlsx) содержит в себе определенный объём данных. В файле имеется столбец "Категория", которая определяет к какой категории относятся конкретные строки данных и к каким шаблонам (файлы B_(101)_file.xlsx, B_(102)_file.xlsx, B_(103)_file.xlsx) будут перекинуты эти данные.

---------------------------------------------------

[EN]

The program is designed to work with large amounts of data in Excel spreadsheets.
Its function is to distribute the data from the feed into templates by category into the appropriate columns.



The principle of operation can be seen on the example of the CharonExcel 0.4.exe program with files from the excels folder. A feed (A.xlsx file) contains a certain amount of data. The file has a "Category" column, which determines to which category specific data lines belong and to which templates (files B_(101)_file.xlsx, B_(102)_file.xlsx, B_(103)_file.xlsx) these data.
